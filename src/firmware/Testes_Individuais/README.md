# Validação: Testes Individuais
Esta pasta contém informações referentes aos códigos-fonte utilizados nos testes individuais dos componentes, bem como os respectivos resultados obtidos. Também foram incluídas pastas contendo capturas do monitor serial, utilizadas para registrar os dados gerados durante os ensaios.

**O objetivo desses testes é apenas validar o funcionamento adequado dos componentes. As atividades de calibração e a implementação da lógica de resolução do labirinto serão desenvolvidas e armazenadas em outro local.**

### Objetivo dos testes
| Componente    | Objetivos dos Testes |
|--------------|----------------------|
| Dual Core    | Verificar a implementação e execução de rotinas em ambos os núcleos do processador. |
| Inerciômetro | Verificar a comunicação com o dispositivo e avaliar a resposta a movimentos. |
| ToF          | Verificar a comunicação com o sensor e avaliar a resposta a diferentes distâncias. |
